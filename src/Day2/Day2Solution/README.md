# Выполненные задания

1. **Transformer**
   - Реализован класс `Transformer` с методом `TransformToWords` для преобразования `System.Double` в словесный формат.
   - Разработаны модульные тесты для проверки метода.

2. **Расширение для System.Double**
   - Реализован метод `Transform`, преобразующий массив чисел типа `double` в массив строк в словесном формате.
   - Разработаны модульные тесты для проверки метода.

3. **Polynomial**
   - Разработан неизменяемый класс `Polynomial` для работы с многочленами одной переменной вещественного типа.
   - Переопределены виртуальные методы `Equals`, `GetHashCode`, `ToString`.
   - Перегружены операторы `+`, `-`, `*`, `==`, `!=`.
   - Разработаны модульные тесты для проверки методов и операторов.
